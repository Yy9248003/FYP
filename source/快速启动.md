# 快速启动指南

## 🚀 快速启动步骤

### 0. database设置 (MySQL)

```bash
# 创建database
mysql -u root -p
CREATE DATABASE db_exam CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 导入database结构和数据
mysql -u root -p db_exam < database/db_exam.sql

# 导入练习数据（可选）
mysql -u root -p db_exam < database/init_practice_data.sql
```

### 1. 启动后端 (Django)

```bash
# 进入后端目录
cd source/server

# 启动Django服务
python manage.py runserver
```

**成功标志**: 看到 "Starting development server at http://127.0.0.1:8000/"

### 2. 启动前端 (Vue)

```bash
# 打开新终端，进入前端目录
cd source/client
npm run serve
# 启动Vue服务


```

**成功标志**: 看到 "App running at: - Local: http://localhost:8080/"

提示: `npm run serve` 为开发模式（热更新，未压缩）。生产部署请使用 `npm run build` 生成静态资源，并通过 Nginx 等静态服务器提供访问。

## 📦 生产环境一键部署

以下脚本用于在全新服务器上快速完成生产部署（包括后端虚拟环境、依赖安装、静态文件收集、前端构建、Nginx 配置与 systemd 自启）。请先准备好 `.env`（参考 `env.example`）。

### Linux (Ubuntu 20.04/22.04+)

```bash
sudo bash deploy/setup_server.sh \
  --project-root /path/to/FYP2025-main \
  --domain your.domain.com \
  --env-file /etc/exam/.env \
  --db-import yes

# 可选：签发 HTTPS 证书
sudo apt-get install -y certbot python3-certbot-nginx
sudo certbot --nginx -d your.domain.com
```

### Windows (PowerShell 以管理员运行)

```powershell
powershell -ExecutionPolicy Bypass -File deploy\setup_server.ps1 `
  -ProjectRoot "C:\\FYP2025-main" `
  -EnvFile "C:\\exam\\.env" `
  -DBImport Yes

# 手动运行（开发/测试）
cd C:\\FYP2025-main\\source\\server
.\.venv\\Scripts\\Activate.ps1
python -m uvicorn server.asgi:application --host 0.0.0.0 --port 8000 --workers 2
```

### 前端生产构建（如需单独执行）

```bash
cd source/client
npm install
npm run build   # 生成 dist 目录
```

### Nginx 与 systemd 示例

- 示例 Nginx 配置: `deploy/nginx.conf`（含 gzip、超时、/api 代理）。
- 示例 systemd 单元: `deploy/exam-gunicorn.service`（含访问/错误日志、graceful-timeout）。

部署要点：
- `.env` 中设置 `DEBUG=False`、`ALLOWED_HOSTS`、`CORS_ALLOWED_ORIGINS`、`CSRF_TRUSTED_ORIGINS`。
- `python manage.py collectstatic --noinput` 后，Nginx 通过 `/static/`、`/media/` 提供文件。
- Windows 上若 `mysqlclient` 构建失败，脚本会自动使用 PyMySQL 兼容。
