# 工具脚本目录

本目录包含各种调试和诊断工具。

## 文件说明

- **debug_jwt.py** - JWT令牌调试工具
- **diagnose_zhipuai.py** - 智谱AI连接诊断工具
- **verify_api_key.py** - API密钥验证工具

## 使用方法

1. **JWT调试**: `python debug_jwt.py`
2. **AI诊断**: `python diagnose_zhipuai.py`
3. **密钥验证**: `python verify_api_key.py`

## 注意事项

- 运行前确保已安装Python依赖
- 某些工具可能需要配置环境变量
- 建议在开发环境中使用这些工具
