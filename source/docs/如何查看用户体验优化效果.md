## 🎨 设计系统查看方法

### 方法1: 查看CSS文件

**文件位置**: `source/client/src/assets/design-system.css`

**内容**:
- CSS变量定义（颜色、字体、间距等）
- 统一组件样式类
- 响应式设计规则

### 方法2: 在浏览器中查看

1. 打开浏览器开发者工具（F12）
2. 查看任意页面的样式
3. 可以看到使用了设计系统的CSS变量和类

**示例**:
```css
/* 查看按钮样式 */
.btn-primary {
  background: var(--primary-gradient);
  /* ... */
}
```

## 📊 图表组件查看方法

### 方法1: 访问数据可视化页面

1. 登录系统（学生账号）
2. 点击左侧菜单 **"数据可视化"**
3. 查看三个图表组件

### 方法2: 查看组件代码

**文件位置**:
- `source/client/src/components/ChartCard.vue` - 通用图表卡片
- `source/client/src/components/StudentProgressChart.vue` - 学习进度图表
- `source/client/src/components/ScoreTrendChart.vue` - 成绩趋势图表
- `source/client/src/components/WrongQuestionStatsChart.vue` - 错题统计图表

### 方法3: 在其他页面中使用

可以在任何页面中导入并使用这些图表组件：

```vue
<template>
  <StudentProgressChart />
</template>

<script>
import StudentProgressChart from '@/components/StudentProgressChart.vue'
export default {
  components: {
    StudentProgressChart
  }
}
</script>
```

## 🛠️ 工具函数查看方法

**文件位置**: `source/client/src/utils/ui-helpers.js`

**包含功能**:
- `showSuccess/Error/Warning/Info` - 提示方法
- `formatTime` - 时间格式化
- `formatRelativeTime` - 相对时间格式化
- `formatFileSize` - 文件大小格式化
- `copyToClipboard` - 复制到剪贴板
- `debounceDecorator/throttleDecorator` - 防抖节流

**使用方式**:
```javascript
import { showSuccess, formatTime } from '@/utils/ui-helpers.js'

// 显示成功提示
showSuccess('操作成功')

// 格式化时间
const formatted = formatTime('2024-01-01 12:00:00')
```
