## 🎯 使用说明

### 消息转发
1. 在消息中心选择要转发的消息
2. 点击"转发"按钮
3. 选择接收者（支持多选）
4. 可选添加转发说明
5. 点击"确认转发"

### 任务结果查看
1. 在任务中心找到已完成的任务
2. 点击"查看结果"按钮
3. 查看任务详细信息（得分、正确率、用时等）
4. 查看答题记录详情

### 个人动态
1. 进入个人动态页面
2. 自动加载考试、练习、任务记录
3. 按时间线展示所有活动
4. 支持按类型和日期筛选
5. 显示统计数据（总活动数、本周活动、本月活动）
